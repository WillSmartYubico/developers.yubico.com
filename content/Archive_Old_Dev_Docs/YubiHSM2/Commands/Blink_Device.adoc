== BLINK DEVICE

Blink the device.

== Description

Blink the LED of the device to identify it.

== Shell Example

Blink the device for 15 seconds:

  yubihsm> blink 0 15

== Protocol Details

=== Command

|===========
|T~c~ = 0x6b
|L~c~ = 1
|V~c~ = S
|===========

S := Seconds to blink for (1 byte)
