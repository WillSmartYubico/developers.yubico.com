== DELETE OBJECT

Delete an Object.

== Description

Delete an Object in the device.

== Shell Example

Delete Asymmetric Key `0x52b6`:

  yubihsm> delete 0 0x52b6 asymmetric-key

== Protocol Details

=== Command

|===============
|T~c~ = 0x58
|L~c~ = 2 + 1
|V~c~ = I \|\| T
|===============

I := link:../Concepts/Object_ID.adoc[Object ID] (2 bytes)

T := link:../Concepts/Object.adoc[Type] (1 byte)
